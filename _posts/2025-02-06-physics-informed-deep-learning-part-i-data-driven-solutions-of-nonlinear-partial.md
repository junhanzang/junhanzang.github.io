**초록**  
본 논문에서는 일반 비선형 편미분 방정식으로 표현되는 임의의 물리 법칙을 준수하면서 지도학습 문제를 해결하도록 학습된 신경망, 즉 **물리 정보 기반 신경망(Physics Informed Neural Networks, PINNs)** 을 소개한다. 이 두 부분으로 구성된 논문에서, 우리는 두 가지 주요 문제—데이터 기반 편미분 방정식 해법과 데이터 기반 편미분 방정식 발견—를 해결하는 맥락에서 진행한 연구 성과를 제시한다. 사용 가능한 데이터의 특성과 구성에 따라, 연속 시간 모델과 이산 시간 모델이라는 두 가지 별개의 알고리즘 클래스를 고안하였으며, 이로써 기저에 존재하는 물리 법칙들을 사전 정보로 자연스럽게 인코딩하는 데이터 효율성이 높은 범용 함수 근사기의 새로운 범주를 형성하였다. 본 1부에서는 이러한 신경망을 활용하여 편미분 방정식의 해를 추론하고, 모든 입력 좌표 및 자유 변수에 대해 완전히 미분 가능한 물리 정보 기반 대리 모델을 얻는 방법을 보여준다.

**키워드:** 데이터 기반 과학 컴퓨팅, 머신 러닝, 예측 모델링, 룽게-쿠타 방법, 비선형 동역학  
**저널:** Journal Name

**1. 서론**  
최근 폭발적인 데이터와 컴퓨팅 자원의 확대로 인해, 머신 러닝과 데이터 분석의 최신 발전은 이미지 인식 [1], 자연어 처리 [2], 인지 과학 [3], 유전체학 [4] 등 다양한 과학 분야에서 혁신적인 성과를 거두고 있다. 그러나 복잡한 물리, 생물학, 공학 시스템을 분석하는 과정에서는 데이터 획득 비용이 매우 높아, 불완전한 정보 하에서 결론을 도출하고 의사 결정을 내려야 하는 어려움에 직면하는 경우가 많다. 이러한 소규모 데이터 환경에서는 딥/컨볼루션/순환 신경망과 같은 최신 머신 러닝 기법들이 견고성이 부족하여 수렴 보장을 제공하지 못하는 한계가 있다.

처음에는 소수의 (잠재적으로 매우 고차원적인) 입력 및 출력 데이터 쌍으로부터 비선형 매핑을 정확하게 식별하도록 딥 러닝 알고리즘을 훈련시키는 과제가 기껏해야 단순해 보일 수 있다. 그러나 물리 및 생물학적 시스템의 모델링과 관련된 많은 경우, 현대 머신 러닝 실무에서 활용되지 않고 있는 방대한 사전 지식이 존재한다. 시스템의 시간 의존적 동역학을 지배하는 체계적인 물리 법칙, 또는 경험적으로 검증된 규칙이나 기타 도메인 전문지식 등이 그러한 예이다. 이와 같은 사전 정보는 정규화 인자로 작용하여 허용 가능한 해의 공간을 관리 가능한 크기로 제한할 수 있으며(예를 들어, 비압축성 유체 역학 문제에서 질량 보존 원칙을 위배하는 비현실적인 흐름 해를 배제함으로써), 학습 알고리즘이 관찰하는 데이터의 정보량을 증폭시켜 소수의 학습 예제만 있을 때에도 올바른 해로 빠르게 수렴하고 우수한 일반화 성능을 발휘하도록 돕는다.

구조화된 사전 정보를 활용하여 데이터 효율적이며 물리 정보를 반영한 학습 기계를 구축하려는 초기 시도들은 이미 최근 연구들 [5, 6, 7]에서 소개되었다. 해당 연구에서는 Gaussian 프로세스 회귀 [8]를 사용하여 주어진 선형 연산자에 최적화된 함수 표현을 고안함으로써, 수학 물리학의 여러 프로토타입 문제에 대해 해를 정확하게 추론하고 불확실성 추정치를 제공하였다. 이후 Raissi 등 [9, 10]은 추론과 시스템 식별의 맥락에서 비선형 문제로의 확장을 제안하였다. 비록 Gaussian 프로세스가 사전 정보를 인코딩하는 데 유연성과 수학적 우아성을 제공하더라도, 비선형 문제를 다루는 과정에서는 두 가지 중요한 한계가 존재한다. 첫째, [9, 10]에서는 시간에 따른 비선형 항들을 국소적으로 선형화해야 했기 때문에 제안된 방법의 적용 범위가 이산 시간 도메인에 한정되었으며, 강한 비선형 영역에서 예측의 정확도가 저하되는 문제가 있었다. 둘째, Gaussian 프로세스 회귀의 베이지안적 특성은 특정 사전 가정을 필요로 하여 모델의 표현 능력을 제한하고, 특히 비선형 문제에서 견고성 및 취약성 문제가 발생할 수 있다 [11].

**1.1 문제 설정 및 기여 요약**  
본 연구에서는 딥 뉴럴 네트워크를 활용하여, 범용 함수 근사기로서의 탁월한 능력을 기반으로 하는 새로운 접근 방식을 취한다 [12]. 이 설정에서는 어떠한 사전 가정, 선형화, 또는 국소 시간 스텝팅 없이도 비선형 문제를 직접적으로 해결할 수 있다. 또한, 과학 컴퓨팅 분야에서 매우 유용하지만 아직 충분히 활용되지 않은 자동 미분 기법 [13]의 발전을 이용하여, 신경망을 입력 좌표와 모델 파라미터에 대해 미분함으로써 물리 정보 기반 신경망을 구성한다. 이 신경망은 관측된 데이터를 지배하는 물리 법칙에서 유래한 대칭성, 불변성, 보존 원칙 등을 반드시 준수하도록 제약된다. 이러한 단순하면서도 강력한 구성 방식은 계산 과학의 광범위한 문제들을 해결할 수 있을 뿐만 아니라, 새로운 데이터 효율적 물리 정보 반영 학습 기계, 편미분 방정식을 위한 새로운 범주의 수치 해법, 그리고 모델 반전 및 시스템 식별을 위한 데이터 기반 접근법 개발로 이어질 잠재적으로 혁신적인 기술을 소개한다.

본 연구의 궁극적인 목표는 수학 물리학 분야에서 오랜 기간 축적된 발전을 딥 러닝에 융합시킴으로써, 모델링 및 계산 분야의 새로운 패러다임의 기초를 마련하는 것이다. 이러한 발전은 데이터 기반 편미분 방정식 해법과 데이터 기반 편미분 방정식 발견이라는 두 가지 주요 문제 범주의 맥락에서 제시된다. 이를 위해, 다음과 같은 일반 형태의 파라미터화된 비선형 편미분 방정식을 고려한다:

### 2.1 예제 (버거스 방정식)

예제로서, 버거스 방정식을 고려해보자. 이 방정식은 유체 역학, 비선형 음향학, 가스 역학, 교통 흐름 등 응용수학의 다양한 분야에서 등장한다 [14]. 버거스 방정식은 기본적인 편미분 방정식으로, 압력 구배 항을 생략함으로써 속도장에 대한 Navier-Stokes 방정식으로부터 유도될 수 있다. 점성 계수의 값이 작을 경우, 버거스 방정식은 고전적인 수치 기법으로 해결하기 어려운 충격(shock) 형성을 야기할 수 있다. 1차원 공간에서, 디리클레 경계 조건과 함께 버거스 방정식은 다음과 같이 표현된다:

![](/assets/images/posts/501/img_2.png)

```
def u(t, x):
    u = neural_net(tf.concat([t, x], 1), weights, biases)
    return u
```

이에 대응하여, 물리 정보를 반영한 신경망 f(t,x)는 다음과 같은 형태를 가진다:

```
def f(t, x):
    u_val = u(t, x)
    u_t = tf.gradients(u_val, t)[0]
    u_x = tf.gradients(u_val, x)[0]
    u_xx = tf.gradients(u_x, x)[0]
    f_val = u_t + u_val * u_x - (0.01 / tf.pi) * u_xx
    return f_val
```
