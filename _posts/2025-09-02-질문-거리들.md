# ? VCA Technology 면접 준비 정리

## 1. 회사/제품 이해

- **VCA Technology**: 영상 분석(Video Content Analytics) 전문 기업, 영국 본사.
- 주요 제품: **VCA Server** (AI 기반 영상 분석 플랫폼).
- 기능: 사람/차량 탐지, 행동 인식(폭력·낙상), HOI 추적, 카운팅, ANPR 등.
- 실제론 YOLO 같은 detection 구조와 유사하지만, tracking, behaviour analysis, rule 기반 event 처리까지 **엔드투엔드 솔루션** 제공.

---

## 2. Framework (일하는 방식)

- **문제 정의 → 데이터 분석 → 모델 설계 → 최적화/배포**
  - 문제 정의: CCTV 환경(저해상도, occlusion, latency) 고려.
  - 데이터: 증강(흑백, rotation, blur, noise, 조명, 날씨 합성 등)으로 다양성 확보.
  - 모델 설계: ViT/ResNet backbone + detection/tracking head(DETR-lite, YOLO류).
  - 최적화: distillation → pruning → quantization(static 중심, 혼합정밀) → ONNX Runtime(OpenVINO).
- **연구와 실제 응용을 연결하는 사이클**을 내 framework로 설명 가능.
