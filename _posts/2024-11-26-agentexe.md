'agent.exe'는 Anthropic의 Claude 3.5 Sonnet 모델의 컴퓨터 제어 기능을 활용하여 사용자의 로컬 컴퓨터를 직접 제어할 수 있게 해주는 간단한 Electron 애플리케이션입니다.

**주요 기능:**

- **Claude 3.5 Sonnet과의 통합:** 이 애플리케이션은 Claude 3.5 Sonnet 모델을 통해 컴퓨터를 제어하며, 이를 위해 Anthropic API 키가 필요합니다.
- **크로스 플랫폼 지원:** 주로 macOS에서 테스트되었지만, Windows와 Linux에서도 이론적으로 작동할 수 있도록 설계되었습니다.
- **간단한 설정:** .env 파일에 Anthropic API 키를 추가하고, npm install 및 npm start 명령어로 애플리케이션을 실행할 수 있습니다.
- **Firefox와의 호환성:** Claude는 Firefox 브라우저를 선호하므로, 원활한 작동을 위해 Firefox를 설치하는 것이 권장됩니다.

이 애플리케이션은 사용자가 Claude 3.5 Sonnet의 컴퓨터 제어 기능을 쉽게 체험할 수 있도록 설계된 프로토타입입니다.
